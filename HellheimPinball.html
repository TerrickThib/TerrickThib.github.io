About

HellHeim Pinball is a game I worked on as a Designer and Programmer it is a Pinball game where the board is made to look like the gates of Hellheim. You use the pattles or in this case the swords to keep the ball in HellHeim as long as you can and score points.

Task

Board Design/Collisons

I created the board layout by using references from other pinball games as well as edited the board so the ball doesnt get stuck and rolls well. I also handled all the colliders in the game so the ball doesnt break out of bounds and so it rolls well.

Gameplay/Ball Movement/Respawning

I worked on gameplay and ball movemment. The overall feel of the game and how the ball moves around the board.

UI Scoring And Respawn

I made it so the UI in the game updates when the ball scores points aswell as when it falls into the drain and loses a ball.
